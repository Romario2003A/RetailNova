# RetailNova

Proyecto final de Base de Datos 2 basado en una empresa ficticia de retail y comercio electronico.

## Contenido

- `sql/`: scripts de SQL Server para OLTP, optimizacion, seguridad, auditoria, DWH y demo.
- `mongodb/`: scripts para generar y consultar eventos digitales en MongoDB.
- `etl/`: proceso ETL multi-fuente para integrar SQL Server, MongoDB y CSV.
- `data/`: datos auxiliares, como campanias comerciales.
- `bi/`: dashboard de Power BI y medidas DAX.
- `docs/`: informe final, presentacion y evidencias.

## Bases principales

- `RetailNova_OLTP`: base transaccional para clientes, pedidos, productos, pagos e inventario.
- `RetailNova_DWH`: Data Warehouse para analisis, Power BI e indicadores.
